# gns3-network-basics

Небольшой учебный проект по базовой сетевой настройке в GNS3.  
Внутри находятся несколько задач, каждая из которых включает топологию, конфигурации устройств и файл проекта `.gns3`.

## Task 1  
Базовая топология с несколькими устройствами.  
Содержит файл проекта и конфигурации устройств.

## Task 2  
Небольшая топология с двумя роутерами и VPCS-хостами.  
Внутри лежат конфиги устройств и лог-файлы.

## Task 3  
Расширенная топология с тремя роутерами.  
Файлы конфигураций находятся в папке `dynamips`.

## Task 4  
Небольшая лабораторная работа с двумя устройствами.  
Включает конфиги роутеров и файл проекта `.gns3`.

---

## Используемые инструменты

- *GNS3*
- *Dynamips*
- *VPCS* 
- Базовые сетевые протоколы (IP, ARP, ICMP)

Проект сделан в учебных целях.
